พัฒนาด้วย ❤️ เพื่อดึงข้อมูลจาก Google Sheets | วิธีการใช้งาน

วิธีการใช้งาน

×

ขั้นตอนการใช้งาน:

  1. ใส่ Google Apps Script URL ที่ Deploy แล้ว
  2. ใส่ API Token ที่ตั้งค่าไว้
  3. เลือก Sheet ที่ต้องการดึงข้อมูล
  4. กำหนดตัวกรองตามต้องการ
  5. คลิกปุ่ม "ดึงข้อมูล"

รูปแบบข้อมูลที่ส่ง:

{
    "token": "your-token",
    "sheetName": "Sheet1",
    "id": "001",
    "options": {
        "limit": 10,
        "offset": 0,
        "filters": {
            "column": "value"
        }
    }
}

หมายเหตุ:

⚠️ Google Apps Script อาจมีปัญหา CORS ดังนั้นควรใช้ JSONP หรือทดสอบด้วย Postman
