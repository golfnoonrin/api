# Google Sheets Data Viewer

เว็บแอพพลิเคชันสำหรับดึงข้อมูลจาก Google Sheets ผ่าน Google Apps Script API

## คุณสมบัติ

- 🔍 ดึงข้อมูลจาก Google Sheets ผ่าน API
- 🔐 รองรับการตรวจสอบ Token
- 📊 แสดงข้อมูลในรูปแบบตาราง
- 🔎 ค้นหาและกรองข้อมูล
- 📄 รองรับการแบ่งหน้า (Pagination)
- 📥 Export ข้อมูลเป็น CSV และ JSON
- 📋 คัดลอกข้อมูล
- 🔄 รีเฟรชข้อมูลอัตโนมัติ
- 📜 ประวัติการเรียก API
- 🌐 รองรับ CORS และ JSONP
